# ATM Simulator

A simple ATM simulator built with HTML, CSS, and JavaScript.

## Features

- Enter PIN and verify user.
- View user details (name, surname, balance).
- Withdraw money if sufficient balance is available.

## Technologies Used

- **HTML** for structure
- **CSS** for styling
- **JavaScript** for functionality
